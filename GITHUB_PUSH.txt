2. Agrega el remoto de GitHub:
   git remote add origin https://github.com/marco-calvar/sistema-comparativo-jacobi-gauss-seidel.git

3. Cambia a rama main:
   git branch -M main

4. Haz push al repositorio:
   git push -u origin main

5. Cuando pida credenciales:
   - Usuario: marco-calvar
   - Contraseña: Tu contraseña de GitHub O token personal

OPCIÓN 2: Desde GitHub CLI (si tienes instalado)

1. Instala GitHub CLI: https://cli.github.com

2. Autentica:

3. Sigue los pasos en pantalla

4. Crea repositorio:
   gh repo create sistema-comparativo-jacobi-gauss-seidel --public --source=. --remote=origin --push

CONFIGURACIÓN TOKEN PERSONAL (SI PIDE CONTRASEÑA)
================================================================================

Si Git pide contraseña en lugar de aceptar tu contraseña de GitHub:

1. Ve a: https://github.com/settings/tokens
2. Click "Generate new token" → "Generate new token (classic)"
3. Nombre: "Git Personal Access Token"
4. Selecciona scope: ☑ repo (Full control)
5. Click "Generate token"
6. COPIA el token (no lo podrás ver después)
7. Cuando Git pida contraseña: Pega el token

Guardalo en lugar seguro (Notepad encriptado, gestor contraseñas, etc.)
